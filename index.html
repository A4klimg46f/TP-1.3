Explicar como funciona a Web e o que é uma página web

Uma página na web funciona primeiramente através de um protocolo WWW para que possa acessar a página. Este protocolo significa: World Wide Web, que é uma aplicação onde podemos verificar a interligação de documentos e páginas através daquilo que chamamos de link.Podemos dizer, de maneira simplificada, que uma página web é um documento renderizado/compilado e acessível a partir de qualquer navegador que tenha acesso a internet. Páginas web compõem a World Wide Web, e que de certa forma são tradidos pelo navegador utilizado para acessar a internet.

Explicar os papéis das linguagens HTML e CSS numa página web

As linguagem HTML (Linguagem de Marcação de Hipertexto) é definida como uma linguagem de marcação e hipertexto, que são estruturadas para estruturação de elementos de uma página como alguns que podemos citar: parágrafos, links, imagens, tabelas, textos, é o bloco de construção mais simples e básico da web. Os documentos HTML são interpretados pelos navegadores. O HTML é utlizado para finalidades estruturais e não para fins funcionais, portanto não pode ser considerada uma linguagem de programação.O HTML não possui nenhuma lógica de programação, não é possível através dela criar estrutras do tipo condicionais como algumas existentes em outras linguagens de programação, e também náo é possível declaração de funções ou execução de cálculos. o Já a linguagem CSS (Cascading Style Sheet) possui um papel diferente da linguagem HTML, ela serve para separar o conteúdo do site de sua apresentação visual. Esta linguagem altera alguns elementos cores, estilos, ou seja, todo o aspecto estético de uma página web. esta linguagem possui uma forte relação com o HTML, sáo linguagens que andam juntas. O CSS permite que se estilize tudo em um arquivo, criando desta forma o estilo separadamente.HTML e CSS se integram.
